# Optimizing-Management-of-Employee-Payroll-Data

This code was created to assist in gathering and averaging employee payroll data.

##Usage

This project can be used as a tracker to create lists of payroll data salaries.  

##Features

The average salary from the data entered can be viewed in the console.  You can also select a random employee from the data entered.
